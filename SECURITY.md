# Security Policy
## Reporting a Vulnerability

We take the security of ESDetect seriously. If you believe you have found a security vulnerability, please report it to us as described below.

### How to Report

Please send a detailed report of the vulnerability to the author via email michalz10100@gmail.com.

Please include the following details in your report:
*   Type of vulnerability (e.g., buffer overflow, privilege escalation, etc.).
*   Full paths of source file(s) related to the manifestation of the vulnerability.
*   Step-by-step instructions to reproduce the issue.
*   Proof-of-concept or exploit code (if available).
*   Impact of the vulnerability.

### Response Timeline

*   We will acknowledge receipt of your report as soon as possible (best effort).
